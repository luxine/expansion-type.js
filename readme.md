# expansion-type.js介绍

#### 本包拓展了JavaScript的基础数据结构，已实现的数据结构如下

#### 1.`栈（Stack）`

#### 2.`队列（Queue）`

#### 3.`双端队列（Deque）`

#### 4.`链表（LinkedListNode）`

##### 后续将逐步拓展更多的数据结构

****

#### 安装

**NPM**
```
NPM install expansion-type.js
```

**yarn**
```
yarn add expansion-type.js
```
****

#### 简介

**本包的源码实现均采用`接口（Interface）-->抽象类（AbstractClass）-->实现类（ImplementationClass）`设计，源码请查看`src`目录下文件**



#### 关于详细的API将在维护完成后给出
